# Ritual Firefox Theme

A Firefox theme based on the latest version of the [Srcery palette](https://github.com/srcery-colors/srcery-palette). It is part of the Ritual desktop shell found [here](https://github.com/roosta/dotfiles).

## Limitations

There are limits to what you're allowed to color, this theme is just comprised
of the [manifest.json](./addon/manifest.json) file, and I've addressed all
available color fields according to [MDN Docs](https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/manifest.json/theme),
but there are things im not allowed to target with a static theme.
